## Requirements

iOS 6.0+

## Installation

JQCollectionViewWaterfallLayout is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'JQCollectionViewWaterfallLayout'
```
